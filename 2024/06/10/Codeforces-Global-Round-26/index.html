C2. Magnitude (Hard Version)

题意: 给定数组 a 和 初始数字 c = 0, 依次遍历 a，每一步可以 c = c + a[i]，或者 c = abs(c + a[i])，求最后获得最大值的方案数

分析: $dp$ 即可，遍历数组的同时遍历上一次个dp 数组中的最大值和最小值的方案数，记录方案数，类似 01背包记录方案书，很容易知道最终答案一定是每一步的最小值或者最大值转移而来
